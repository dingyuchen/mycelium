# Classification Learning

Let's define some terms

Instances
: Input, set of inputs
: Vectors of values

Concept
: Function that maps inputs to outputs

Target Concept
: Actual Answer

Hypothesis (Class)
: Set of concepts that you are willing to entertain

Sample
: Training Set
: Values paired with label

Candidate
: A concept under test

Testing Set
: The final test
